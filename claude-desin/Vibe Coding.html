Curriculum

5단계 커리큘럼

한 단계씩 따라오면, 마지막엔 작동하는 AI 에이전트가 손에 쥐어집니다.

  1. STEP 01

    바이브 코딩의 이해

    AI와 함께 일하는 새로운 방식, "Vibe Coding"의 개념과 워크플로우를 익힙니다.

    • AI 협업의 기본 멘탈 모델
    • 업무 자동화의 4단계 사고법
  2. STEP 02

    Claude Code 가이드

    자연어로 코드를 만들고 수정하는 Claude Code 사용법을 처음부터 배웁니다.

    • 설치부터 첫 프로젝트까지
    • 요청을 잘 쓰는 프롬프트 패턴
  3. STEP 03

    파일 정리 자동화하기

    매일 반복하는 다운로드·문서 정리를 스크립트 한 줄로 끝냅니다.

    • 폴더 자동 분류 스크립트
    • 주간 보고서 자동 생성
  4. STEP 04

    랜딩 페이지 제작

    HTML/CSS를 몰라도 브랜드에 맞는 랜딩페이지를 디자인하고 실제로 배포합니다.

    • 섹션 구조 설계 → 반응형 적용
    • 실제 도메인에 배포까지
  5. STEP 05

    OpenAI 에이전트 만들기

    OpenAI API로 내 업무 맥락을 이해하는 챗봇·에이전트를 직접 빌드합니다.

    • OpenAI API 연결 & 프롬프트 설계
    • 대화 기록과 도구(tool) 호출 구현
